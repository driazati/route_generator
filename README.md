# Delivery Routes

## Installation

Get code

```bash
git clone https://github.com/driazati/route_generator.git
cd route_generator
pip install -r requirements.txt
```

Set up Bing Maps API key

Create a file called `secret.py` and add the key:
```bash
echo "bing_api_key = '<YOUR API KEY>' > secret.py"
```

## Run

```bash
python main.py
```
